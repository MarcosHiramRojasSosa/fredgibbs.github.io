Digital Mapping Examples

  • Show ABQ Bike Paths with D3 and mouseover event
  • Show NM Counties as paths and cities as circles (both from geoJSON files) with D3, with mouseover event
  • Show NM City boundaries using Leaflet, OSM tiles, and loading geoJSON data
  • Load a US Map with custom tiles from Mapbox styled from web interface or Tilemill
  • Load a US Map with tiles from Stamen maps and loading a KML file exported from Google Maps Engine

Notes

All associated files (data, javascript, etc) can be viewed by finding and clicking on the file names in the HTML source code.

Notice that many of these files load maps (or configuration parameters) in a slightly different way to provide a variety of examples of how one can initialize and configure a map.

If you don't understand something in any of the code, or you need an example of something else, tweet at me (@fredgibbs) and use the #digmaps hashtag so everyone can follow the thread.
